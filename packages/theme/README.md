# @ibiz-template/theme

基于scss，定义样式的全局变量、scss函数指令、scss混合指令等，定义UI层的样式书写规范。

## 📂 项目结构

```javascript
|──  theme
    ├── style                                                               样式文件夹
       ├── global.scss                                                      导出样式
       ├── settings     		
       │   ├── config.scss  	                                            样式配置
       │   └── var.scss			                                            样式规范
       └── tools
           ├── css-var.scss		                                            scss变量
           ├── function.scss                                                scss函数
           ├── mixin.scss   	                                            scss混合
           └── utils.scss   	                                            scss工具
```
