# @ibiz-template/runtime

提供前端应用运行所需的核心逻辑能力，自上而下包含AppHub、应用、视图、部件、部件项等对象的逻辑层能力封装、应用服务模块（包含异步行为、应用权限、动态存储、代码表、计数器等核心服务）、数据逻辑编排、界面行为、界面逻辑编排、视图逻辑、动态替换预置组件及插件管理等功能模块。
